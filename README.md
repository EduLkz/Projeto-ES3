Para rodar o front-end é necessario entrar no diretorio ./frontend-es3 e rodar o comando "npm run start", isso erá abrir o servidor https. 
O frontend é hospedado no https://localhost:3000

Para rodar o back-end é necessario entrar no diretorio ./backend-es3, ativar o ambiente virtual com o comando "source ./es3-venv/bin/activate" e então "python3 app.py".
O backend é hospedado no http://localhost:5000
